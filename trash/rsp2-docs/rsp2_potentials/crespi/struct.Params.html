Fields

delta: f64

Transverse distance scaling factor. Units are Angstroms.

lambda: f64

Distance scaling factor for the repulsive potential. Units are inverse Angstroms.

A: f64

Multiplicative factor for the attractive potential. Units are eV.

z0: f64

“Convenience scaling factor.” Units are Angstroms.

C: f64

Offset constant used in repulsive potential. Units are eV.

C2N: [f64; 3]

Multiplicative constants used in repulsive potential. Contains C0, C2, C4. Units are eV.

cutoff_begin: f64

Distance at which the Kolmogorov-Crespi potential starts to switch to zero. Units are Angstroms.

cutoff_transition_dist: Option<f64>

How far it takes for the Kolmogorov-Crespi to switch to zero after the cutoff distance. Units are Angstroms.

A value of None can be used to simulate Lammps’ behavior of a sharp cutoff. In this case, the value will be offset slightly for C0 continuity.

Constants used for calculation of the Kolmogorov-Crespi potential.

These match the values used by default by the implementation of kolmogorov/crespi/z in Lammps, which is scaled to Lammps’ rebo’s bond length.

pub fn compute(&self, r_ij: V3, normal_i: V3, normal_j: V3) -> Output[src]

Calculate the Kolmogorov-Crespi potential.

Accepts the delta vector between two carbon atoms r_ij as well as the unit normal vectors for each atom, n_i and n_j. Returns the potential as well as the gradient of the potential with respect to r_ij, n_i, and n_j.

For the total potential of a system, one should only sum over undirected bonds. (see FracBond::is_canonical).
