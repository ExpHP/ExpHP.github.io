Represents an objective function with an analytically known gradient, to be minimized by conjugate gradient.

You can use a fn(&[f64]) -> Result<(f64, Vec<f64>), E> closure, but note that you must annotate the closure argument enough for Rust to see the higher-ranked lifetime. (i.e. |pos: &_| ...).
