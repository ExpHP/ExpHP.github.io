Module rsp2_array_types::mat[−][src]

Small fixed-size matrix types, compatible with V2/V3/V4

This library primarily uses a row-based formalism; matrices are conceptually understood to be containers of row-vectors. This formalism is most useful when most vectors used are row vectors (in which case most matrix-vector multiplication has the matrix on the right)
