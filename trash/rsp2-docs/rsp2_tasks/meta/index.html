Module rsp2_tasks::meta[−][src]

Site metadata

High-level code carts site metadata around in the form of HLists of Rc<[T]> (or similar, like Option<Rc<[T]>>). The use of Rc eliminates concern about ownership (it is distinctly un-fun to turn an Option<Vec<T>> into an Option<&[T]), and allows this module to get away with a very small number of extension traits. (where a reasonably ergonomic API that can handle differences in ownership would require at least half a dozen)
